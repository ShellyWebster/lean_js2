Display

Properti CSS display digunakan untuk menentukan bagaimana elemen HTML ditampilkan di halaman web. Properti ini memiliki beberapa nilai yang dapat digunakan untuk mengubah cara elemen ditampilkan, seperti block, inline, inline-block, none, dan lain-lain.
